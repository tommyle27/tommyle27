Snake

Language:

C# via Unity

About:

This was the first recreation of a video game to completion! It was a really fun project that helped me learn some pixel art skills along the way while designing the apples the snake eats. It is also the first game that is fully developed in code, (available in my github). This project taught me how to implement spawning of objects in random locations, and design the game for a balanced and fun challenge for the player, as the snake increases in size at a steady rate. Not too slow or fast where the player may become bored or upset with the game. This process was a done through trial and error, playing through the game to find the sweet spot for a well balanced game of snake.
